## Description

Process your images with this simple API. More details in the [documentation](https://image-cropper-api.onrender.com/doc).

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
